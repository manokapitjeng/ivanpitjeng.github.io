SOC Analyst Project

Tools Used

  1. Oracle VM VirtualBox Manager: For creating and managing virtual machines (VMs).
  2. Splunk Server: For log analysis and monitoring.
  3. Splunk Universal Forwarder: For data forwarding to Splunk.
  4. Sysmon: Endpoint monitoring on Windows machines.
  5. Crowbar: Used to simulate brute force attacks.
  6. Atomic Red Team (ART): Used for security testing and validation.
  7. Microsoft Windows Event Logs: Analyzed in Splunk for security monitoring.
  8. Windows Server 2022: Operating system used for Active Directory Domain Services setup.
  9. Ubuntu Server: Used as a Splunk server in the lab setup.
  10. Kali Linux: Used as an attacker machine in the lab setup.

Lessons Learned

  • Setting up VMs (Windows 10, Kali Linux, Windows Server, Ubuntu Server) in Oracle VM VirtualBox.
  • Configuring IP addresses, NAT Networks for VMs.
  • Troubleshooting network connectivity (ping, DNS settings).
  • Installing Splunk Server and Universal Forwarder.
  • Gained knowledge of penetration testing techniques using Kali Linux.
  • Installing Sysmon for endpoint monitoring.
  • Developed skills in managing Active Directory and configuring user authentication systems.
  • Using Crowbar for brute force attacks.
  • Using Atomic Red Team (ART) to simulate tests.
  • Analyzing security logs (event codes 4625, 4624) in Splunk.
  • Joining Windows machines to a domain.
  • PowerShell scripting for tasks (Invoke-WebRequest, Set-ExecutionPolicy).
